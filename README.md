# Currency Converter Privacy Policy

このリポジトリは Currency Converter（モバイル/ウェブ対応）のプライバシーポリシーを、英語と日本語で提供する単一 HTML ページです。GitHub Pages でそのまま公開できます。

## ファイル構成
- `index.html` — スタイル・スクリプト・本文をひとつにまとめたページ（英語/日本語切替ボタン付き）

## ローカルで確認
```bash
open index.html
```
ブラウザでページが開き、言語トグルやリンクの見栄えを確認できます。

## 公開手順（GitHub Pages）
1. このフォルダを GitHub リポジトリにプッシュ
2. GitHub の Settings → Pages で Branch を `main`（または `master`）、Folder を `/root` に設定
3. 保存後に表示される URL が公開先になります
